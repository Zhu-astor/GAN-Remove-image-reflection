反光處理算法:

1.訓練 條件GAN(生成對抗網路)，主要分成生成器以及判別器的部分:
	i.  生成器:讀入條件圖像(具反光的照片)，嘗試生成去除反光的照片(引導生成特定類型的輸出)
	ii. 判別器:嘗試區分生成圖像跟真實圖像之區別(配對數據，每張反光都有對應的無反光圖片)
		   保留條件GAN訓練好的生成器模型，獨立出來使用
2.獨立使用生成器模型:
	i. 該模型後續可以直接輸入 真實反光的圖片，生成無反光圖片(模型生成的，準確率我們在第一步驟要訓練，透過判別器去進行校正)

影像辨識:

1.訓練博物館展品的影像辨識模型
	(跟Teachable Machine同個概念，但我們要訓練得更好一點，ex:數據	增強)
2.訓練好的模型用來辨識 前面GAN輸入真實反光圖片生成的無反光圖片
	(就是簡單的影像辨識，只是我們辨識的圖片會先經過反光處理而已)
